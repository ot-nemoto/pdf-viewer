# プロダクト定義

## 目的

ローカルの PDF ファイルを、インストール不要でブラウザ上に素早く表示するビュワーを提供する。
ファイルをサーバーへ送らずクライアント側だけで処理することで、手軽さとプライバシーを両立する。

## 対象ユーザー

- 手元の PDF をさっと開いて確認したい人
- 資料を一定間隔で自動送りしながら閲覧したい人（掲示・レビュー・読み合わせ用途）
- 専用アプリを入れず、ブラウザだけで完結させたい人

## 提供価値

- **導入ゼロ**: URL を開くだけ。インストール・ログイン不要
- **プライバシー**: PDF はブラウザ内で処理され、どこにもアップロードされない
- **軽快な操作**: ドラッグ＆ドロップで開き、キーボードでページ送り、自動送りにも対応

## 成功指標

| 指標 | 目安 |
|------|------|
| PDF を開いて初ページが表示されるまで | ドロップから数秒以内 |
| 主要操作の学習コスト | 説明なしでページ送り・ズーム・自動送りに到達できる |
| クライアント完結率 | 100%（サーバー通信を伴うファイル処理は行わない） |

## スコープ外（現時点）

- PDF の編集・注釈・保存
- 複数ファイルのタブ管理
- テキスト検索・サムネイル一覧
- サーバー保存・共有リンク生成
