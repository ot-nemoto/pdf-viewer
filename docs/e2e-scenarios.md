# E2E / 手動確認シナリオ

UI を含む変更は、以下のシナリオで手動動作確認を行う。確認は[公開環境](https://ot-nemoto.github.io/pdf-viewer/)またはローカル（`npm run dev`）で実施する。

## 1. ファイル読み込み

| # | 手順 | 確認観点 | 期待値 |
|---|------|---------|--------|
| 1-1 | PDF をドロップ領域にドラッグ＆ドロップ | 読み込み | 初ページが表示され、ツールバーが出る |
| 1-2 | ドロップ案内をクリックしファイル選択で PDF を開く | 読み込み | 1-1 と同様に表示される |
| 1-3 | PDF 以外（例: .txt）をドロップ | エラー | エラーメッセージが表示され、表示状態にならない |
| 1-4 | ドラッグ中にドロップ領域へカーソルを重ねる | フィードバック | 領域がハイライトされる |

## 2. ページ操作

| # | 手順 | 確認観点 | 期待値 |
|---|------|---------|--------|
| 2-1 | ツールバーの `›` を押す | ページ送り | 次ページへ進み、`n / N` が更新される |
| 2-2 | 1 ページ目で `‹` を押す | 境界 | 1 ページ未満に進まない（ボタンは disabled） |
| 2-3 | 最終ページで `›` を押す | 境界 | 最終ページを超えない（ボタンは disabled） |
| 2-4 | `→` / `←` キーを押す | キーボード | 次 / 前ページへ移動する |
| 2-5 | `PageDown` / `PageUp` キーを押す | キーボード | 次 / 前ページへ移動する |

## 3. ズーム

| # | 手順 | 確認観点 | 期待値 |
|---|------|---------|--------|
| 3-1 | `＋` を複数回押す | 拡大 | 最大 300% まで拡大し、それ以上増えない |
| 3-2 | `−` を複数回押す | 縮小 | 最小 50% まで縮小し、それ以上減らない |
| 3-3 | 現在倍率ボタン（例: `120%`）をクリック | リセット | 100% に戻る |

## 4. 自動ページ送り

| # | 手順 | 確認観点 | 期待値 |
|---|------|---------|--------|
| 4-1 | 間隔を選び `▶ 自動送り` を押す | 再生 | 選んだ間隔ごとに次ページへ自動遷移する |
| 4-2 | 再生中に `⏸ 停止` を押す | 停止 | 自動遷移が止まる |
| 4-3 | 最終ページまで自動再生 | 自動停止 | 最終ページで再生が自動停止する |
| 4-4 | 最終ページで `▶ 自動送り` を押す | 先頭復帰 | 先頭ページに戻ってから再生が始まる |

## 5. 終了

| # | 手順 | 確認観点 | 期待値 |
|---|------|---------|--------|
| 5-1 | `✕ 閉じる` を押す | リセット | ドロップ待ち（Empty）状態に戻る |
| 5-2 | 閉じた後に別の PDF を開く | 再読込 | 新しい PDF が 1 ページ目・等倍・停止状態で表示される |
